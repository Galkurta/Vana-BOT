# Vana BOT
- Support multi account
- Auto tap
- Auto task
- Register Vana Data Hero [here](https://t.me/VanaDataHeroBot/VanaDataHero?startapp=6944804952)

## Installation

```
git clone https://github.com/Galkurta/Vana-BOT
```

```
cd Vana-BOT
```

```
npm install
```

## Usage
### Edit data.txt

```
nano data.txt
```

### Run

```
node main.js
```
